# HexaMatching

### Toy project for hexa 3 matching game.


## <구현 완료 항목>

* 드랍 로직 : 새로운 블럭 들이 빈자리를 찾아 위에서 부터 흘러내려 오는 로직

* 매칭 조건 : 각 세 축의 직선 상 3개의 블럭이 모일 경우

* 스코어 : 각 블록 당 20점


## <개발 환경>

* Unity
	* Editor Version : 2019.1.1f1
	* Platform : Android
